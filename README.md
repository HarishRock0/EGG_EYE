# EGG_EYE

## Overview
This project analyzes EEG data from a neuroheadset, using the dataset from Kaggle: [EEG Neuroheadset Dataset](https://www.kaggle.com/datasets/ppb00x/eeg-neuroheadset).

## Dataset
- **Source:** Kaggle ([link](https://www.kaggle.com/datasets/ppb00x/eeg-neuroheadset))
- **File:** eeg-headset.csv
- The dataset contains EEG signals recorded from a neuroheadset, suitable for machine learning and signal processing tasks.

## Project Structure
- `eeg-headset.csv`: Raw EEG data from Kaggle
- `model.ipynb`: Jupyter notebook for data analysis, preprocessing, and modeling

## Usage
1. Clone the repository:
   ```bash
   git clone https://github.com/HarishRock0/EGG_EYE.git
   ```
2. Open `model.ipynb` in Jupyter Notebook or VS Code.
3. Follow the notebook for data loading, preprocessing, and model training steps.

## Requirements
- Python 3.8+
- Jupyter Notebook
- Common packages: pandas, numpy, scikit-learn, matplotlib (install as needed)
